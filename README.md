# wx-love-msg
微信推送给对象的每天天气和早安问候
## 环境准备
备注：如果使用已经编译好的exe就不需要准备这些环境
- Springboot 2.6.4
- java 1.8
- maven 3.8

## 依赖项
- 聚合数据天气预报API
- 聚合数据心灵鸡汤API
- 测试的微信公众号

## 原理
 从第三方api网站(本案例使用的是聚合数据的)获取天气预报和心灵鸡汤文案，这个网站每天可以免费调30次，足够用了。获取到这些数据后程序按照微信模板消息的规范组装好，利用微信公众号的模板消息接口推送出去，被推送的人需要先关注公众号
